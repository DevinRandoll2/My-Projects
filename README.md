# My Projects

## Nusha Interpreter
- **Tech:** Java
- **Description:** A fully custom interpreter for the “Nusha” logic‑puzzle language, implemented in Java. The system tokenizes and parses indentation‑sensitive syntax, builds an abstract syntax tree, and evaluates declarative rules using a backtracking constraint solver.
Nusha programs describe structured entities (like Authors, Places, Animals) and the interpreter computes valid story assignments by enforcing uniqueness constraints, multi‑clause conditions, and logical implications. This project demonstrates strong understanding of compilers, language design, recursion, and algorithmic reasoning.
- **Status:** Completed
